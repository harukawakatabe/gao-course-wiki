# 高玮课程 LLM Wiki 建设计划

本目录用于规划和建设一个面向个人长期使用的 LLM Wiki。当前阶段只初始化建设计划，不一次性固定完整目录结构。

## 当前定位

高玮课程 LLM Wiki 不是单门课笔记，也不是单纯的资料备份目录。它的目标是把高玮老师的课程资料逐步编译成一个可阅读、可追溯、可持续维护、可供 Agent 使用的个人知识库。

## 初始原则

- 原始资料和编译后的 wiki 分开管理。
- 课程视角、知识视角、应用视角分开管理。
- 大文件默认不进入 git，只在来源卡片中登记路径、摘要、校验信息和处理状态。
- 面向人读的页面和面向 Agent 的索引都要保留。
- 每次重要摄入、结构调整或结论沉淀都要留下日志。

## 第一阶段目标

第一阶段不追求完整整理全部课程，而是跑通一个最小可维护闭环：

1. 建立来源卡片 schema。
2. 对商业逻辑课文字稿做来源盘点。
3. 选取 5 到 10 篇讲次生成课程层页面。
4. 从样本中抽取概念、框架和案例。
5. 形成第一批跨课程 wiki 页面。
6. 建立 AI 上下文索引和维护 lint 规则。
